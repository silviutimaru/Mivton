## 🚀 Quick Start

### Prerequisites
- Node.js 18+ installed
- Git installed

### Local Development
```bash
# Navigate to project directory
cd ~/Desktop/Mivton

# Install dependencies
npm install

# Start development server
npm run dev

# Visit http://localhost:3000
```

### Production Deployment
```bash
# Start production server
npm start
```
